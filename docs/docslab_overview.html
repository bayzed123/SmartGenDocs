Overview

The SmartGen Documentation Hub serves as the centralized knowledge base, API reference platform, and developer user guide for the SmartGen QR code generation tool ecosystem built by Connect With Bayezid.

Far from a standard static site repository, this project implements a modern, GitOps-driven workflow that abstracts away deployment complexities. It bridges the gap between technical writing and automated publishing, allowing contributors to write raw Markdown and let a robust automation engine handle the rest.

Key Architectural Pillars

  • Continuous Deployment (GitOps): Powered by an automated GitHub Actions pipeline. Any changes merged into the production branch automatically trigger a rebuild, validation, and instantaneous deployment.
  • Dynamic Data Aggregation: Utilizes custom Python scripts within the /scripts directory to dynamically pull and aggregate documentation guidelines and API definitions from external codebases, ensuring content never falls out of sync.
  • Programmatic Branding & Media: Features dynamic Open Graph (OG) card generation. Using Python styling libraries (pillow and cairosvg), the pipeline automatically builds custom, high-resolution social sharing images for every individual page on the fly.
  • Tailored User Experience: Built on the cutting-edge MkDocs Material framework, enhanced with bespoke HTML/CSS overrides and layouts to deliver a highly readable, responsive, and professional documentation portal.
