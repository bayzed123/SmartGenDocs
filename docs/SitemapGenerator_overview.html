Overview

The Auto Sitemap Generator is a robust, developer-friendly backend service and crawler designed to automate the discovery and mapping of web properties. Built on a Node.js architecture, this tool helps developers and SEO specialists systematically index websites, respect search engine boundaries, and generate valid, SEO-compliant sitemap.xml files with zero manual hassle.

Why It Matters

Modern search engines rely on accurate, structured sitemaps to index web pages efficiently. However, maintaining up-to-date sitemaps for dynamic, growing, or JavaScript-heavy websites is a constant challenge. This project solves that problem by combining a systematic traversal engine with modern headless browser technology to generate real-time, search-engine-ready sitemaps.

Key Architectural Pillars

  • Intelligent BFS Crawling Engine: The crawler employs a Breadth-First Search (BFS) algorithm to traverse domains level-by-level. This ensures that high-value, top-level directory pages are discovered and prioritized first, mirroring how search engine bots naturally navigate a site.
  • Modern Web Ready (Puppeteer-Powered): Unlike basic HTML parsers that fail on dynamic content, this generator leverages Puppeteer to run a headless browser instance. This allows it to easily execute JavaScript, rendering and mapping single-page applications (SPAs) and highly interactive web platforms.
  • Strict Compliance & Safety: Good crawler etiquette is baked directly into the core. The engine automatically fetches and parses a domain's robots.txt file, ensuring it strictly respects exclusion rules and avoids crawling restricted directories.
  • Infinite Loop Prevention: Using highly efficient in-memory state tracking, the system maintains a secure queue of pending URLs and visited states, completely eliminating the risk of infinite loops caused by circular internal linking structures.
  • Dual-Output Deliverables: Upon completion, the tool produces a standard-compliant, production-ready sitemap.xml alongside structured JSON files (links.json and progress.json) for easier analytics, custom reporting, or debugging.
  • Automated Workflow Integration: With an included GitHub Workflows configuration (crawler.yml), the system is primed for CI/CD environments—allowing you to set up automated, scheduled runs (cron jobs) to keep your sitemaps perpetually updated without manual intervention.

Whether you run it locally as a CLI tool, integrate it into your deployment pipelines via GitHub Actions, or use the interactive frontend dashboard, the Auto Sitemap Generator provides a reliable, production-ready solution to keep your site's SEO architecture in perfect shape.
